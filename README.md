# Top US Sites 2026

A beautifully designed, responsive single-page website showcasing the most visited websites in the United States for 2026.

## Features

- **18 ranked websites** across 6 categories
- **Real traffic data** from Semrush Analytics (April 2026)
- **Category filtering** — Search, Social, E-Commerce, AI, Content
- **Live search** with instant results
- **Dark / Light mode** with system preference detection
- **Responsive design** — mobile, tablet, and desktop
- **Animated cards** with staggered entrance effects

## Data Source

All rankings and metrics are sourced from Semrush Traffic Analytics. Data reflects April 2026 monthly US visit figures.

## Tech Stack

- Pure HTML, CSS, and JavaScript
- No frameworks, no build step
- Zero dependencies
- Deployed via GitHub Pages

## Local Development

Just open `index.html` in your browser — no server needed.
